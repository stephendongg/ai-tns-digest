# AI Trust & Safety Digest

Minimal daily briefing built for one job: help me stay current on AI trust, safety, and governance.

[View live ->](https://stephendongg.github.io/ai-tns-digest/)

## How it works

Pulls recent trust-and-safety relevant posts from labs, evaluators, standards bodies, and policy organizations -> keeps only fresh items from the last 5 days -> filters for alignment, evaluations, governance, security, privacy, red teaming, and safeguards -> uses GPT-5.4 to turn them into one short daily brief while including as many materially important fresh links as the day warrants, rather than stopping at five by default.

## Output

- One lead summary
- A variable-length set of high-signal trust-and-safety links
- A single-page GitHub Pages site with a lightweight archive of previous editions

## Tech

Python - OpenAI API - RSS/Atom feeds - GitHub Actions - GitHub Pages
